NBA 2021-22 Season Stat API

Hello! Welcome to the NBA Stats API for the 2021-22 Season! On this API you can get the stats from every player that played this season and compare utilize it in your program. You can either gather all of the players information or you can select specific players from the data that you want to get into your program.
